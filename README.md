# eretz-ir-6565

Eretz-Ir is a modern, interactive Hebrew version of the classic word game 'Categories' (also known as 'Scattergories' or 'Country-City'), designed for online multiplayer and solo play. The game challenges players to quickly think of Hebrew words starting with a random letter across categories like countries, cities, animals, and foods, with real-time scoring, leaderboards, and a sleek dark-themed interface to enhance engagement and social competition.

## Tech Stack
Next.js, Node.js with Socket.io, PostgreSQL, Tailwind CSS

## Setup
Run `docker-compose up` to start the infrastructure.
